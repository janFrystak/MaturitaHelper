1 Tvorba webu – obecně a klient–server

Webové stránky se tvoří pomocí tří základních technologií: HTML (struktura) · CSS (vzhled) · JavaScript (chování/interaktivita). Fungují na principu klient–server.

HTML Struktura stránky — co na stránce je (nadpisy, odstavce, obrázky, tabulky)
CSS Vzhled a rozvržení — jak stránka vypadá (barvy, fonty, pozice)
JavaScript Chování stránky — co se děje při interakci uživatele (kliknutí, formuláře, animace)

Princip klient–server

Klient = webový prohlížeč (Chrome, Firefox, Edge…). Odesílá požadavek na server pomocí protokolu HTTP nebo HTTPS.
Server = počítač poskytující webové stránky. Přijme požadavek a vrátí HTML dokument (+ CSS, JS, obrázky).
HTTP vs. HTTPS

HTTP (HyperText Transfer Protocol) = nešifrovaný přenos. HTTPS = šifrovaný pomocí TLS/SSL — bezpečná verze, nutná pro přihlašování a platby. Prohlížeče zobrazují zámek 🔒.

2 Jazyk HTML

HTML (HyperText Markup Language) = značkovací jazyk pro popis struktury webové stránky. Není programovací jazyk — popisuje co na stránce je, ne jak se chová. Používá tagy (značky).

Struktura HTML dokumentu

<!DOCTYPE html>
<html lang="cs">
  <head>
    <meta charset="UTF-8">
    <title>Název stránky</title>
  </head>
  <body>
    Viditelný obsah stránky
  </body>
</html>
TagFunkce
<!DOCTYPE html>Určuje verzi HTML (aktuálně HTML5)
<html>Kořenový element — obaluje celý dokument
<head>Metadata stránky — neviditelná část (title, charset, CSS odkaz)
<meta charset="UTF-8">Znaková sada — umožňuje diakritiku
<title>Název stránky zobrazený v záložce prohlížeče
<body>Viditelný obsah stránky

Základní HTML elementy

Nadpisy h1–h6 
<h1>Hlavní nadpis</h1>
<h2>Podnadpis</h2>
<h3>Třetí úroveň</h3>

h1 má nejvyšší důležitost — jen jeden na stránce. Důležité pro SEO.

Odstavce a text 
<p>Odstavec textu.</p>
<strong>Tučné</strong>
<em>Kurzíva</em>
<br>  <!-- zalomení řádku -->
Odkaz 
<a href="https://example.com">Text</a>
<a href="page.html">Interní</a>
<a href="..." target="_blank">Nová záložka</a>

href = cílová adresa (URL nebo relativní cesta)

Obrázek 
<img src="obrazek.jpg"
     alt="Popis obrázku"
     width="300">

src = cesta · alt = alternativní text (přístupnost, SEO)

Seznamy 
<ul>          <!-- nečíslovaný -->
  <li>HTML</li>
  <li>CSS</li>
</ul>
<ol>          <!-- číslovaný -->
  <li>První</li>
</ol>
Tabulka 
<table>
  <tr>
    <th>Hlavička</th>
  </tr>
  <tr>
    <td>Buňka</td>
  </tr>
</table>

Kontejnery a sémantické elementy

Obecné kontejnery
  • <div> — blokový element pro rozdělení stránky. Začíná na novém řádku, zabírá celou šířku.
  • <span> — řádkový element pro část textu. Nezačíná na novém řádku.
Sémantické elementy (HTML5)
  • <header> — hlavička stránky
  • <nav> — navigace
  • <main> — hlavní obsah
  • <section> — tematická sekce
  • <article> — samostatný článek
  • <aside> — boční panel
  • <footer> — patička stránky

Výhody: přehlednost · lepší SEO · přístupnost (screen readery)

HTML formuláře

Slouží pro zadávání dat uživatelem — přihlášení, kontaktní formulář, vyhledávání.

<form action="zpracuj.php" method="POST">
  <input type="text" placeholder="Jméno" name="jmeno">
  <input type="email" placeholder="Email">
  <input type="password" placeholder="Heslo">
  <input type="checkbox" id="souhlas">
  <textarea rows="4">Zpráva</textarea>
  <select>
    <option>Možnost 1</option>
  </select>
  <button type="submit">Odeslat</button>
</form>
action — URL zpracování method GET/POST input type: text · email · password · number · checkbox · radio · file · date

3 Jazyk CSS – vzhled a rozvržení

CSS (Cascading Style Sheets) = stylovací jazyk pro oddělení vzhledu od struktury. Kaskáda = pravidla se dědí a překrývají podle priority (specifičnost selektoru).

3 způsoby zápisu CSS

1. Inline styl Přímo v HTML tagu. Nejnižší priorita pro opětovné použití, nejvyšší specifičnost. Používat výjimečně. 
<p style="color:red;">Text</p>
2. Interní styl V tagu <style> uvnitř <head>. Platí pro celou stránku. 
<style>
  p { color: red; }
</style>
3. Externí CSS soubor Odkaz na soubor .css. Nejlepší praxe — jeden soubor pro celý web. 
<link rel="stylesheet"
      href="style.css">

CSS Box Model

Každý HTML element je obdélníkový „box" složený z vrstev (od středu ven):

ContentSamotný obsah (text, obrázek). Určuje width a height.
PaddingVnitřní odsazení — prostor mezi obsahem a ohraničením. padding: 10px;
BorderOhraničení. border: 1px solid black;
MarginVnější odsazení — prostor mimo element, mezi sousedy. margin: auto;
* { box-sizing: border-box; } /* padding a border se počítají do width — doporučeno vždy nastavit */

Základní CSS vlastnosti

Text a font
  • color — barva textu
  • font-size — velikost (px, em, rem, %)
  • font-family — druh písma
  • font-weight — tloušťka (bold, 700)
  • text-align — zarovnání (left, center, right)
  • line-height — výška řádku
  • text-decoration — podtržení, přeškrtnutí
Pozadí
  • background-color — barva pozadí
  • background-image — obrázek na pozadí
  • background-size: cover — pokryje celou oblast
  • background-repeat — opakování
  • background-position — pozice
Rozměry a zobrazení
  • width / height — šířka / výška
  • max-width — maximální šířka
  • display — block · inline · flex · grid · none
  • visibility: hidden — skryje, ale zachová místo
  • overflow — jak řešit přetečení obsahu
  • opacity — průhlednost (0–1)

CSS Flexbox — jednorozměrné rozvržení

Flexbox rozmísťuje elementy v jednom směru (řádek nebo sloupec). Ideální pro navigaci, galerie, zarovnávání prvků.

/* Na rodičovský kontejner */
.container {
  display: flex;
  flex-direction: row;        /* row (výchozí) | column */
  justify-content: center;    /* zarovnání na hlavní ose: flex-start | center | flex-end | space-between | space-around */
  align-items: center;        /* zarovnání na křížové ose: stretch | center | flex-start | flex-end */
  flex-wrap: wrap;            /* zalamování na více řádků */
  gap: 16px;                  /* mezera mezi prvky */
}

/* Na dítě */
.item {
  flex: 1;                    /* kolik místa zabere (poměr) */
  flex-grow: 1;               /* smí růst */
  flex-shrink: 0;             /* nesmí se smrštit */
}

CSS Grid — dvourozměrné rozvržení

Grid rozmísťuje elementy v mřížce (řádky × sloupce). Ideální pro celkové rozložení stránky.

/* Na rodičovský kontejner */
.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;  /* 3 sloupce — střední dvakrát širší */
  grid-template-rows: auto;
  gap: 16px;                            /* mezera mezi buňkami */
}

/* Položka zabírající více sloupců */
.header {
  grid-column: 1 / -1;       /* od 1. do posledního sloupce */
  grid-row: 1;
}

Responzivní design a media queries

Web musí fungovat na počítači · tabletu · mobilu. Přizpůsobení pomocí media queries — podmíněné CSS pravidla podle šířky obrazovky.

/* Mobile-first přístup — nejdřív styly pro mobil, pak přidávej pro větší */
body { font-size: 14px; }

@media (min-width: 768px) {    /* tablet a větší */
  body { font-size: 16px; }
  .container { display: flex; }
}

@media (min-width: 1200px) {   /* desktop */
  .container { max-width: 1140px; margin: auto; }
}

/* Nebo: max-width přístup — nejdřív desktop, pak zmenšuj */
@media (max-width: 600px) {
  .nav { flex-direction: column; }
}
procentuální šířky max-width Flexbox a Grid media queries rem/em místo px pro text viewport meta tag 
<meta name="viewport" content="width=device-width, initial-scale=1.0">

CSS proměnné (Custom Properties)

:root {
  --primary-color: #0C447C;
  --font-size-base: 16px;
}

p {
  color: var(--primary-color);
  font-size: var(--font-size-base);
}

CSS animace a přechody

/* Transition — plynulý přechod při změně stavu */
button {
  background: blue;
  transition: background 0.3s ease, transform 0.2s;
}
button:hover {
  background: darkblue;
  transform: scale(1.05);
}

/* Animation — opakující se animace */
@keyframes pulse {
  0%   { opacity: 1; }
  50%  { opacity: 0.5; }
  100% { opacity: 1; }
}
.loading {
  animation: pulse 1.5s infinite;
}

4 JavaScript – chování stránky

JavaScript = skriptovací jazyk běžící v prohlížeči. Přidává stránce interaktivitu — reaguje na akce uživatele, mění obsah stránky, komunikuje se serverem. Interpretovaný jazyk s dynamickým typováním.

Vložení do HTML 
<!-- Inline -->
<button onclick="alert('Klik!')">Klikni</button>

<!-- Interní -->
<script>
  console.log("Ahoj");
</script>

<!-- Externí soubor -->
<script src="script.js"></script>
Základní operace s DOM 
// Výběr elementu
document.getElementById("id")
document.querySelector(".trida")

// Změna obsahu
el.textContent = "Nový text";
el.innerHTML = "<b>Tučný</b>";

// Změna stylu
el.style.color = "red";

// Událost
btn.addEventListener("click", () => {
  alert("Kliknuto!");
});

5 Formáty XML a JSON

XML — eXtensible Markup Language Značkovací jazyk pro přenos a ukládání dat. Vlastní (uživatelsky definované) značky. Přísná syntaxe — každý tag musí být uzavřen. Hierarchická struktura. 
<student>
  <jmeno>Jan</jmeno>
  <trida>4A</trida>
</student>

Použití: konfigurace programů · RSS · starší webové služby · výměna dat

JSON — JavaScript Object Notation Lehký textový formát pro přenos dat. Nejpoužívanější formát moderních webových API. Dvojice klíč–hodnota. 
{
  "jmeno": "Jan",
  "trida": "4A",
  "znamky": [1, 2, 1]
}

Použití: komunikace klient–server · REST API · moderní weby

6 Upload webových stránek

Upload = nahrání souborů webu na webový server, aby byly dostupné přes URL adresu z celého internetu.

FTPFile Transfer Protocol — nešifrovaný přenos souborů. Klient: FileZilla.
SFTPSecure FTP — šifrovaný přenos přes SSH. Bezpečnější varianta FTP.
Webhostingové rozhraníSprávce souborů přímo v prohlížeči — jednoduchá správa bez FTP klienta.
Git / GitHub PagesNasazení webu pushnutím do repozitáře — moderní přístup, verzování zdarma.

7 Redakční systémy (CMS) a Wiki

CMS — Content Management System Umožňuje správu webu bez programování. Obsah, šablony a pluginy přes grafické rozhraní.
  • ✓ Jednoduchá správa obsahu
  • ✓ Hotové šablony a pluginy
  • ✓ Administrace přes prohlížeč
  • ✗ Nižší flexibilita než čistý kód

Příklady: WordPress · Joomla · Drupal

Wiki systémy Umožňují společnou tvorbu obsahu více uživateli přímo v prohlížeči.
  • Editace přes prohlížeč bez instalace SW
  • Historie změn — lze vrátit libovolnou verzi
  • Verzování — kdo, kdy, co změnil

Příklad: MediaWiki (základ Wikipedie)
